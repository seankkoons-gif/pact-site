Get started

Verifier-first path: install, run the demo, verify the pack, open the Evidence Viewer. No SDK required to try verification.

3-command quickstart

Install dependencies, run the canonical demo, then verify the Auditor Pack and open the viewer.

# 1. Install
pnpm install --frozen-lockfile
# or: npm install

# 2. Run demo + verify pack
bash demo/h5-golden/run_all.sh
bash design_partner_bundle/verify_all.sh

# 3. Open Evidence Viewer (local)
pnpm --filter @pact/evidence-viewer dev

After step 3, drag and drop an Auditor Pack ZIP into the viewer to inspect verdict, timeline, DBL judgment, and export GC/Insurer PDFs.

Verifier commands

Offline verification from the command line. Use these after you have an Auditor Pack or transcript.

# Verify full Auditor Pack (ZIP)
pact-verifier auditor-pack-verify --zip <pack.zip>

# Replay transcript (determinism check)
pact-verifier replay --transcript <transcript.json>

# Run DBL judgment on transcript
pact-verifier judge --transcript <transcript.json>

Exact subcommands may vary by release; see the repo for the latest CLI.
